# savedata
savedata for games

## Commands:

## Epic Games

```
git clone --no-checkout --depth 1 https://github.com/alphaxleonidas/savedata
cd savedata
git sparse-checkout init
git sparse-checkout set "EpicGames/<GameName>"
```
```
git checkout
```

## Steam Games

```
git clone --no-checkout --depth 1 https://github.com/alphaxleonidas/savedata
cd savedata
git sparse-checkout init
git sparse-checkout set "Steam/<GameName>"
```
```
git checkout
```
